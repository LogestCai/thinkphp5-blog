## 博客系统

### 开始

* 安装  

  `git clone https://github.com/LogestCai/thinkphp5-blog`  

* 配置

  将项目目录下的blog.sql 的数据库文件还原至mysql数据库。

  修改`config`目录下的database.php文件，将里面对应参数设置为自己实际环境参数。

* 账户

  用户名：admin 密码：123456   登录后可自行修改。
